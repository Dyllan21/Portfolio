Applying Styles:

The add Class method can be used to fill in the defined style sheets onto all of the matched elements. You can also specify multiple classes that are separated by space

Code:

            <html>
            <head>
            <title>the title<title>
            <script type="text/javascript"
            src="/jquery/jquery-1.3.2.min.js"><script>
            <script type="text/javascript" language="javascript">
            $(document).ready(function() {
            $("em").addClass("selected");
            $("#myid").addClass("highlight");
            });
            <script>
            <style>
                .selected { color:red; }
                .highlight { background:yellow; }
                <style>
            <head>
            <body>
                <em title="Bold and Brave">This is first paragraph.<em>
                <p id="myid">This is second paragraph.<p>
            <body>
            <html>

Display:

the title This is first paragraph.

This is second paragraph.

atrr(key,func) Method:

It sets a single property to a computed value on all the matched elements. The key is the name of the property to set. The func is returning the value to set function would've had one argument which is the index of the current element.

Code:

            <html>
            <head>
            <title>The Selecter Example<title>
            <script type="text/javascript" src="/jquery/jquery-1.3.2.min.js">
            <script>
            <script type="text/javascript" language="javascript">
                $(document).ready(function() {
                    $("table").attr("border", function(arr) {
                        return arr;
                    })
                });
            <script>
            <head>
            <body>
                        
            <table>
            <tr><td>This is first table<td><tr>
            <table>
                        
            <table>

        <body>
        <html>

hasClass(class) Method:

This method returns true if the class that's specified is present on of the sets of the matched elements or otherwise it can return as false.

Code:

            <html>
                <head>
                    <title>The Selecter Example<title>
                    <script type="text/javascript" 
                src="http://ajax.googleapis.com/ajax/libs/jquery/2.1.3/jquery.min.js">
                    <script>
                
                <script type="text/javascript" language="javascript">
                    $(document).ready(function() {
                        $("#result1").text( $("p#pid1").hasClass("red") );
                        $("#result2").text( $("p#pid2").hasClass("red") );
                    });
                <script>
                
                <style>
                    .red { color:red; }
                    .green { color:green; }
                <style>
                <head>
                
                <body>
                
                    <p class="red" id="pid1">This is first paragraph.<p>
                    <p class="green" id="pid2">This is second paragraph.<p>
                
                    <div id="result1"><div>
                    <div id="result2"><div>
                
                <body>
                
            <html>

Display:

text() Method:

The text() method is where a text that contents of all the matched elements are combined. This can work for XML and the XHTML documents.

Code:

                <html>
                    <head>
                    <title>The Selecter Example<title>
                    <script type="text/javascript" src="/jquery/jquery-1.3.2.min.js">
                    <script>
                    
                    <script type="text/javascript" language="javascript">
                        $(document).ready(function() {
                            var content = $("p#pid1").text();
                            $("p#pid2").html(content);
                        });
                        <script>
                        <style>
                            .red { color:red; }
                            .green { color:green; }
                        <style>
                        
                        <head>
                        <body>
                        
                        <p class="green" id="pid1">This is <i>first paragraph<i>.<p>
                        <p class="red" id="pid2">This is second paragraph.<p>
                        
                        <body>
                        <html>
